# Typing Speed Test

![Image](https://github.com/user-attachments/assets/3589a47a-9b27-4d71-9b5b-8af06bef0b56)
A simplest typing app created to interact with a Blackberry Passport keyboard due to unavailability of Monkeytype, Clicks Quest, etc. because of the limited web-browser

```python

```

## Requirements
https://github.com/sw7ft/BerryCore/ is required

Tested in Term49\
font_size = 50 is recommended

## Installation
Download the repo
```
wget https://github.com/mercedesov/type-test/archive/refs/heads/main.zip
```
Unzip `unzip main.py`\
Run `python app.py`
